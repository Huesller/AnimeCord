# Proposta do Projeto

Criar um "Discord" para podermos interagir sobre um tema de escolha que mais se agrada.

## Descrição

Projeto esta sendo desenvolvido para treinamento de utilização do framework react, está sendo desenvolvido na imersão React da Alura.

## Primeiro dia

Iniciamos um projeto Next.js;
Criamos components com React usando CSS in JS;
Vimos a estrutura inicial de um projeto Next.js;
Passamos propriedades para components;
Feito deploy na Vercel.
